^p^ + password: send sign up password to server
^u^ + username: send sign up username to server
*p* + password: send login password to server
*u* + username: send login username to server

><|: end of a message
^o^ + name : request to server open a new window connecting to the user with that name
&t& + text: text message sent from user to server
name + $%^: write to server with specified name
^o>: send back to confirm that user is still online
)*( + username: send to server signaling that user has closed his program
)-& + name: request past messages with this user
$#* + user + $%^ + extension + ?|? + byte of image: send message image to server
